Startupx plugin will add custom made elementor elements to elementor page builder to customize sites with ease.

== Description ==

Startupx Toolkit is a powerful and advanced all in one Elementor addons that help you to create a beautiful website with ease. It offers many customizations and unique styling options to enhance the look and feel of every element on your website.

Build a beautiful website with Startupx Toolkit in less time without any coding skills.

= Requirements =

* WordPress 4.7 or later.
* [CodeGearThemes Startupx Themes](https://www.codegearthemes.com/products/startupx).

== Installation ==

1. Install the plugin either via the WordPress.org plugin directory, or by uploading the files to your server (in the /wp-content/plugins/ directory).
2. Activate the Startupx Toolkit plugin through the 'Plugins' menu in WordPress.

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of Startupx Toolkit, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "Startupx Toolkit" and click Search Plugins. Once you’ve found our  plugin you can view details about it such as the the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

= Manual installation =

The manual installation method involves downloading our plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
